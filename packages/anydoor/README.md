## @hunyuan/anydoor
> 静态资源服务器
## 用法
```
$ npm install -g @hunyuan/anydoor
$ anydoor

anydoor [option]

Options:
  -p, --port      端口号                  [default: "9000"]
  -h, --hostname  host               [default: "127.0.0.1"]
  -d, --root      root path        [default: process.cwd()]
      --help      Show help                       [boolean]
  -v, --version   Show version number             [boolean]
```
